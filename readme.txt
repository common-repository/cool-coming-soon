Simple, Super Cool Coming Soon and Maintenance plugin with date, time countdown and auto-launch feature. It's fully customizable and provides maximum display controls.

== Description ==

The Cool Coming Soon plugin allows you to quickly & easily set up a Coming Soon Page. It's recommended to start new website with coming soon page, and highly recommended to use during maintenance. Admin sometime breaks website, during maintenance or customization tasks. The broken site will leave bad impression on your visitors, and may leak sensitive information such as internal directory structures. You can prevent that, by using this small utility.

This plugin will add a page called "Cool Coming Soon" under Settings. You can navigate there and enable/disable maintenance mode and can customize front-end data along with display options.

= Main Features: =
* Easy to use
* Full control over display (Title, Logo and description)
* Auto Launch Feature
* Several Cool Built-in Backgrounds
* Super Fast

This plugin only uses jQuery, and does not rely on any third part library/package. The main purpose of this plugin was to make simple, super fast and easy to operate plugin.


= When we need this plugin? =

* A Simple Coming Soon Page
* Launching New Site
* Maintenance Mode
* Make website offline for any reason / bug

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Cool Coming Soon'
3. Click on 'Install Now'
4. Activate Cool Coming Soon from your Plugins page.

= From WordPress.org =

1. Download Cool Coming Soon.
2. Unzip plugin.
2. Upload the 'Cool Coming Soon' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Cool Coming Soon from your Plugins page.

= Usage =

1. Go to Settings > Cool Coming Soon to customize coming soon page.
2. On the 'Data' tab/page, you can update coming soon message including (Title, Description, Logo, Background Image etc)
3. On the 'Display Options' tab/page, you can enable/disable front-end items.

= Uninstalling: =

1. In the Admin Panel, go to "Plugins" and deactivate the plugin.
2. Delete the plugin from Plugins page.


== Frequently Asked Questions ==

= Having issues with Cool Coming Soon plugin? =

Post your question on support forum for this plugin and I will try to answer as soon as I can.

= Who can see the website while it’s under construction? =
Admin and other logged in users can see the actual website while you are working on the site. Visitor or Logged out users will see the coming soon page

= Does Cool Coming Soon make any changes to the database? =

Yes. It stores configuration data in wordpress database. And it is removed, upon plugin removal.
